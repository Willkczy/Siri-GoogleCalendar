# Siri Calendar Webhook – Vercel 版本

此專案為 Siri Shortcuts 提供 Google Calendar 今日事件查詢服務，部署於 Vercel。

📁 專案結構簡述：
siri-calendar/
├── api/today-events.js      # API 主要邏輯
├── .env                     # GOOGLE_CREDENTIALS
├── package.json
├── .gitignore
└── README.vercel.md

🚀 快速部署：

1. npm install -g vercel
2. vercel login
3. vercel
